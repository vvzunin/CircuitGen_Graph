# CircuitGen Graph v1.7.0

Release date: 2026-05-22  
Previous release: [v1.6.0](https://hub.mos.ru/circuitgen/CircuitGen_Graph/-/tags/v1.6.0)

## English

### Highlights

- **Verilog import** into the graph via `GraphReader` / `OrientedGraph::readVerilog` (**lorina** submodule).
- **Subgraph external RTL**: bind a Verilog file to subgraph vertices; parse `parameter` / `localparam`.
- **Documentation portal on Synology NAS**: shared CircuitGen portal, `manifest.json` schema v2, version channels `versions/main/` and `versions/<tag>/`; bilingual deploy guide in `docs/en|ru/DEPLOY.md`.
- **Testbench generator** improvements and Icarus-based golden tests.
- **Topic guides** (GraphML, Verilog import, subgraphs, testbench) and refreshed `docs/en|ru` index.
- **Logging** unified through `CG_LOG_*` and **easylogging++**.
- Bus support and `oneBitVerilog` for gates; sequential logic updates.

### CI and packaging

- Multi-OS GitLab CI: NAS docs deploy after Doxygen; Docker release images tagged like the Git release.
- Portal links to [hub.mos.ru](https://hub.mos.ru) repositories and per-OS Docker images.

### Upgrade notes

- Requires **CMake 3.26+**, C++17, and updated **lorina** submodule (`git submodule update --init`).
- Consumers of Generator/Parameters should use compatible versions from the CircuitGen ecosystem.

Full changelog: [docs/en/CHANGELOG.md](../en/CHANGELOG.md) (section **1.7.0**).

## Русский

### Основное

- **Импорт Verilog** в граф (`GraphReader`, `OrientedGraph::readVerilog`, подмодуль **lorina**).
- **Внешний RTL для подграфов**: привязка Verilog-файла к вершинам подграфа; разбор `parameter` / `localparam`.
- **Портал документации на Synology NAS**: общий портал CircuitGen, `manifest.json` схема v2, каналы `versions/main/` и `versions/<tag>/`; руководство по деплою в `docs/en|ru/DEPLOY.md`.
- Улучшения **генератора testbench** и golden-тесты с Icarus.
- **Тематические руководства** (GraphML, Verilog, подграфы, testbench) и обновлённое оглавление `docs/ru|en`.
- **Логирование** через макросы `CG_LOG_*` и **easylogging++**.
- Шины и `oneBitVerilog` для вентилей; доработки sequential.

### CI и поставка

- GitLab CI на нескольких ОС: деплой документации на NAS после Doxygen; Docker release-образы с тем же тегом, что и Git-релиз.
- В портале — ссылки на репозитории [hub.mos.ru](https://hub.mos.ru) и Docker-образы по ОС.

### Замечания по обновлению

- Нужны **CMake 3.26+**, C++17 и актуальный подмодуль **lorina** (`git submodule update --init`).
- Для связки с Generator/Parameters используйте совместимые версии экосистемы CircuitGen.

Полный журнал: [docs/ru/CHANGELOG.md](../ru/CHANGELOG.md) (раздел **1.7.0**).
