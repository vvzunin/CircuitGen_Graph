std::string CG_Graph::GraphVertexSequential::toVerilog ( ) const
overridevirtual

toVerilog generates a string in Verilog format for the current vertex, representing the valve according to its type and input connections.

If a vertex has no input connections, an empty string is returned. If any input connection is invalid, an exception is thrown.

Returns
A Verilog format string for the current vertex
Exceptions
std::invalid_argumentif any input connection is invalid
