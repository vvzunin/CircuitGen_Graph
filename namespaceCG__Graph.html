Подробное описание

class GraphVertexInput Represents a vertex in a directed graph that serves as an input vertex.

It inherits from the GraphVertexBase class and extends its functionality to work with input vertices

Аргументы
i_baseGraphA pointer to the oriented graph to which this vertex belongs. It can be null ptr, which indicates the absence of a base graph
i_typeVertex type (VertexTypes enumeration). By default, it is set to VertexTypes::input, indicating that this is the input vertex
i_nameThe name of the vertex provided during creation. This is a string of type std::string that identifies a vertex *‍/
